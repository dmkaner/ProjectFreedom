Postsploit Remote Access Trojan

PostSploit is an advanced remote access trojan that uses a postgresql database to execute commands on a remote machine. This approach is extremely effective because of how unconventional it is. Unlike normal socket based RATs, PostSploit does not require port forwarding because of the way it is built. PostSploit uses an approach we like to call the "middleman" approach, instead of normal peer to peer communication, it connects to a database which does all the heavy lifting in terms of data storage and transfer. This also allows for easy anonymity, if the victim script were to fall into the wrong hands, it would be difficult for it to be traced back to the attacker machine.
